### Belangrijke opmerking

Dit project is niet bedoeld om lokaal te runnen. Het fungeert uitsluitend als brug en referentie naar mijn eigen code.

De daadwerkelijke implementatie van de code bevindt zich in de Social Deal repository binnen Bitbucket. Deze repository is niet openbaar en kan daarom niet gedeeld worden, waardoor het niet mogelijk is om de volledige werkende versie van het project in deze omgeving te tonen.

Daarnaast is dit project op het laatste moment toegevoegd, met als doel de aangeleverde bestanden zo volledig en representatief mogelijk te maken. Hierdoor is het niet uitvoerbaar zonder toegang tot de oorspronkelijke repository.
